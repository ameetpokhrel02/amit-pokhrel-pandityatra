# Monthly Audit Checklist

_Last updated: January 28, 2026_

## 1. Real Data Usage Audit
- [ ] Review backend and frontend for any mock/fake data usage
- [ ] Check API endpoints and DB queries for real PostgreSQL data
- [ ] Validate all user, pandit, and admin flows use production data

## 2. System Integrations Audit
- [ ] Verify payment, booking, chat, video, and notification integrations
- [ ] Check logs for failed integrations or API errors
- [ ] Ensure all third-party services are operational

## 3. User Feedback Collection
- [ ] Review user feedback channels (support, email, in-app)
- [ ] Summarize common issues/feature requests
- [ ] Track feedback on new/updated flows

## 4. Automation & Reminders
- [ ] Confirm recurring calendar reminder is set for monthly audit
- [ ] Update this checklist as needed
- [ ] Assign audit responsibility for this cycle

## 5. Reporting
- [ ] Document findings in shared location (repo, Notion, etc.)
- [ ] List action items and assign owners for any issues found

---

_This checklist should be reviewed and updated at the start of each month._
